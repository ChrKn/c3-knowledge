# Caesar 3 Modding Tools

A most likely rather incomplete list of Caesar 3 modding tools. 

## EngConverter

A tool created by **bvschaik** to convert the Caesar 3 language files (`.eng`) 
to `.xml` and vice versa. Can be used to change in-game textes. It can be found
[here](https://github.com/bvschaik/citybuilding-tools).
